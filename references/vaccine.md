# 疫苗接种记录

## 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 接种日期 | YYYY-MM-DD | 2024-03-15 |
| 疫苗名称 | 全称或简称 | 乙型肝炎疫苗 |
| 剂次 | 第几针 | 第1针 |
| 接种机构 | 医院/社区卫生服务中心 | XX社区卫生服务中心 |
| 下次接种时间 | YYYY-MM-DD 或"满X月" | 2024-04-15 |
| 反应/备注 | 接种后观察情况 | 注射部位轻微红肿，次日消退 |

## 追加记录模板

```
---
💉 {接种日期} | {疫苗名称}（{剂次}）| {接种机构}
   下次：{下次接种时间} | 反应：{反应/备注}
```

示例：
```
---
💉 2024-03-15 | 乙型肝炎疫苗（第1针）| XX社区卫生服务中心
   下次：2024-04-15（满1月）| 反应：无明显不适
```

## 更新待接种计划

接种完成后，将对应计划表中的 ⬜ 改为 ✅，并填入实际日期：
```
| 出生 | 乙肝（第1针）、卡介苗 | ✅ 2024-03-01 |
```

## 查询场景

- "上次打了什么针？" → 找最新一条接种记录
- "下次什么时候打疫苗？" → 找所有"下次"字段，找最近的一个
- "卡介苗打了没有？" → 搜索疫苗名称关键词
