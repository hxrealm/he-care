# 每日奶量记录

## 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 日期 | YYYY-MM-DD | 2024-03-15 |
| 时间 | HH:MM | 08:30 |
| 类型 | 母乳 / 配方奶 / 混合 | 配方奶 |
| 量 | 毫升数（ml） | 120ml |
| 备注 | 可选，如"喝完后吐奶少量" | - |

## 追加记录模板

追加到笔记时，使用以下格式（一次一条）：

```
---
📅 {日期} {时间} | {类型} | {量}ml | {备注}
```

示例：
```
---
📅 2024-03-15 08:30 | 配方奶 | 120ml | 喝得很顺利
```

## 查询场景

- "今天喝了多少？" → 搜索今日日期的所有奶量条目，汇总
- "最近奶量怎么样？" → 读取笔记最近 7 天记录，统计平均值
- "上次喂奶是几点？" → 找最新一条记录的时间

## 常见问题提示

- 若用户没有说时间，默认填写当前时间
- 若用户没有说类型，询问"是母乳还是配方奶？"
- 单次喂奶量参考：新生儿 30-60ml，1月+ 60-90ml，3月+ 90-120ml，6月+ 120-180ml
