# 户口档案

## 字段说明

档案为静态信息，通常只需填写一次，后续按需更新。

### 基本信息
- 姓名、昵称、性别、出生日期、出生地点
- 出生体重、出生身长、血型（如已知）

### 证件信息
- 出生医学证明编号
- 户口本信息（户号、页码）
- 身份证号（通常在1岁左右办理）
- 护照号（如有）
- 医保卡号

### 家长信息
- 父母姓名、联系方式
- 家庭住址

## 更新模板

更新档案时，追加说明变更内容：

```
---
📋 {更新日期} 更新
   变更字段：{字段名}
   新值：{新内容}
   备注：{原因/说明}
```

示例：
```
---
📋 2024-06-01 更新
   变更字段：身份证号
   新值：110101202403010001
   备注：6月办理完成
```

## 查询场景

- "宝宝出生体重是多少？" → 读取基本信息
- "宝宝的医保卡号是多少？" → 读取证件信息
- "宝宝是什么血型？" → 搜索"血型"字段

## 安全提示

此笔记包含宝宝的敏感个人信息（身份证号等），建议在 IMA 中设置访问权限，不对外分享此笔记链接。
