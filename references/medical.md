# 就医记录

## 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 就医日期 | YYYY-MM-DD | 2024-03-15 |
| 医院/科室 | 机构名 + 科室 | XX儿童医院 儿科门诊 |
| 主诉症状 | 就医原因 | 发烧38.5℃，咳嗽2天 |
| 诊断结果 | 医生诊断 | 急性上呼吸道感染 |
| 处方用药 | 药名、剂量、频次 | 布洛芬混悬液 5ml 发热时服，连花清瘟颗粒 1袋 3次/日 |
| 复诊时间 | 若有 | 3天后复查 |
| 备注 | 医嘱、注意事项等 | 多喝水，避免着凉 |

## 追加记录模板

```
---
🏥 {就医日期} | {医院/科室}
   症状：{主诉症状}
   诊断：{诊断结果}
   用药：{处方用药}
   复诊：{复诊时间}
   备注：{备注}
```

示例：
```
---
🏥 2024-03-15 | XX儿童医院 儿科门诊
   症状：发烧38.5℃，咳嗽2天
   诊断：急性上呼吸道感染
   用药：布洛芬混悬液 5ml 发热时服；连花清瘟颗粒 1袋 3次/日
   复诊：3天后（2024-03-18）
   备注：多喝水，避免着凉
```

## 查询场景

- "宝宝上次看病是什么时候？" → 找最新一条就医记录
- "之前用过什么药？" → 搜索"用药"字段
- "有没有过敏史？" → 搜索"过敏"关键词
- "需要复诊了吗？" → 查找所有复诊时间字段，与当前日期比较
