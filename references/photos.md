# 成长照片记录

## 两种记录方式

### 方式一：文字成长记录（notes 模块）

记录每次测量数据和里程碑事件，追加到"📸 成长照片打卡"笔记。

**字段说明：**

| 字段 | 说明 | 示例 |
|------|------|------|
| 日期 | YYYY-MM-DD | 2024-03-15 |
| 月龄 | X月X天 | 3月2天 |
| 身高 | cm | 62.5cm |
| 体重 | kg | 6.2kg |
| 头围 | cm（可选） | 40cm |
| 描述 | 里程碑/特别记录 | 第一次翻身成功！ |

**追加模板：**
```
---
📸 {日期}（{月龄}）
   身高：{身高} | 体重：{体重} | 头围：{头围}
   记录：{描述}
```

示例：
```
---
📸 2024-03-15（3月2天）
   身高：62.5cm | 体重：6.2kg | 头围：40cm
   记录：第一次翻身成功！左翻右都会了，超开心 🎉
```

### 方式二：照片上传（knowledge-base 模块）

将照片上传到 IMA 知识库。使用 ima-skill 的 knowledge-base 模块：
1. 使用 `create_media` 接口上传图片文件
2. 使用 `add_knowledge` 接口将图片关联到知识库

上传时建议在文件名中包含日期和月龄，便于检索：
- 格式：`YYYYMMDD_X月龄_描述.jpg`
- 示例：`20240315_3月_第一次翻身.jpg`

## 查询场景

- "宝宝现在多高多重？" → 读取最新一条测量记录
- "宝宝3个月的时候多重？" → 搜索"3月"月龄记录
- "第一次翻身是什么时候？" → 搜索"翻身"关键词
- "宝宝成长曲线怎么样？" → 读取所有测量记录，汇总身高体重变化
