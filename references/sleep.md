# 睡眠记录

## 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 日期 | YYYY-MM-DD | 2024-03-15 |
| 入睡时间 | HH:MM | 21:00 |
| 醒来时间 | HH:MM | 06:30 |
| 时长 | 自动计算 | 9.5h |
| 类型 | 夜觉 / 午睡 / 小憩 | 夜觉 |
| 备注 | 可选，如"夜醒2次" | 夜醒1次，约03:00喂奶后继续睡 |

## 追加记录模板

```
---
😴 {日期} | {入睡时间}→{醒来时间}（{时长}）| {类型} | {备注}
```

示例：
```
---
😴 2024-03-15 | 21:00→06:30（9.5h）| 夜觉 | 夜醒1次，03:00喂奶后继续睡
```

## 时长计算规则

- 跨午夜：(24:00 - 入睡时间) + 醒来时间
- 未跨午夜：醒来时间 - 入睡时间
- 精确到 0.5h

## 查询场景

- "宝宝今晚睡了多久？" → 查今日夜觉记录
- "最近睡眠规律吗？" → 读取最近7天记录，统计平均时长和夜醒次数
- "午睡睡了多久？" → 查今日午睡记录

## 常见提示

- 若用户只说"入睡了"，记录入睡时间，醒来时间待后续补充
- 新生儿每日总睡眠参考：14-17h；3-6月：12-15h；6-12月：12-14h
